# Captain Aminul Haque IT Institute (UI Demo)

Inspired UI layout: Programming Hero-like modern colorful landing + smooth animations.
This is a **front-end only** project (no backend). You can plug any API later.

## Tech
- Vite + React + TypeScript
- TailwindCSS
- Framer Motion
- React Router

## Run
```bash
npm install
npm run dev
```

## Build
```bash
npm run build
npm run preview
```

## Customize
- Site text/data: `src/data/siteData.ts`
- Banner/Poster images: `src/assets/*`
- Theme colors: `src/styles.css`
